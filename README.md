# Задание 2. Шардирование

## Как запустить

Запускаем mongodb и приложение

```shell
cd ./mongo-sharding/ 
docker compose up -d

```

Запускаем процесс инициализации шардов и заполняем mongodb данными

```shell
./scripts/mongo-init.sh
```
